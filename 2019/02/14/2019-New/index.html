2019-New
2019-02-14

距离考研结束，自闭一个半月结束。
本来我建立这个博客是准备整理一些项目的，结果莫名的觉得我的一片文章一定要写一篇自己心路历程的文章，告诉自己你到底需要什么，想要什么，该怎么去做。

  • 考研之路

嗯，没错我想考PKU。虽然2019年惨败，说自己不努力也说不出口，说自己努力也说不出口。可是不论旁边的人说什么我都想再坚持一年，我想考上北大。

不仅仅是名校光环，也是为了自己的理想和梦想。大一到大四做过菜鸟APP,入门初级WEB，会简单的python.我好像什么都会一点，却好像又什么都不会。曾经有个朋友好心的劝过我，你还不如去培训班学一年，你这样半吊子。什么都不会更加的吃亏。

可是我毫不在意。因为我以为我会一战上北大，然后从事自己的喜欢的工作，可以稍微比以前的自己幸运一点。可是好像不是这么一回事。

要说朋友的劝说，看到现在的结果我会后悔吗？

答案肯定是否定的。

  • 机器学习

我想考上名校，其实不仅仅是为了以后的前途。是为了实现自己的。说来可笑，我尝试过swift,java,html,c#,python。虽然真的没什么运用的特别好的，可是不后悔。因为大三的时候接触到了机器学习。

我喜欢富于美感的东西，我可以学习UI。可是我终究不想和其他人一样，盲目的根据模板的东西制作东西。因为未来说不定机器也可以的。

我写APP, 我写网页，我写系统。

我都怕未来的某一天我会被取代，即时不是机器人取代我。我也会害怕我那时候丧失了热情，我已经老了，我害怕失去。

更重要的是，我学习了机器学习，发现最简单的KNN可以实际用于人脸识别。我惊叹于数学的美妙，我感叹计算机的强大。

我喜欢收集比赛的数据资料，我喜欢关注每一个建立模型的背后的算法。我夜深人静的时候会止不住的回想，回想算法的奇妙，感叹人类的伟大。

我会想，我以后有一天也想到一个超级棒的算法，解决工业级的问题。

哇塞，真的太帅了！

这应该是我会选考北大的初衷吧。

  • 考研结束后

考完研结束后，我总会想，如果我考一个普通211应该是一件很简单的事情吧。可是我这个总是不服输。

因为想去更大的城市，遇见更好的老师， 成为出色的人。

这一个月我如同败家犬一般，在家里躺着像一个咸鱼。

我觉得自己能够出色，却不知道自己的心态才是关键。

前几天和港大的朋友聊天，忽然一下明白了自己。对，我现在就是弱小又如何，一个真正强大的人在任何困境下，应该需要保持的是自己的本心。

我想学习机器学习，我不想因为自己学历太差不敢投那个大公司。为什么那一个人不能是我呢？

我不比别人差呀。至少我大学四年每天都充实着，快乐的。

我考不上研，我也想在工作领域离机器学习近一点。

  • [ ] 未来的展望

    昨天晚上又转辗反侧。

    我在想人生应该是真的有一种叫 阶级 的东西存在吧。

看恋爱节目《恋梦空间》，其中有两个嘉宾让我印象很深刻。

一个男生本科北理，硕士美国TOP20的名校，工作于硅谷。另一个男生本科复旦，硕士TOP5名校，定居新加坡，有数百万的项目研究，应研究又会定期回中国交流，又称为了复旦的教授。

第一个男生是我2年前幻想过 我最好的结局。我感觉我的人生最多只能到达他这样，甚至到他这样可能还要十足的运气。

然而，我翻看朋友圈，我可能又要被现实打击。

我原来的高中同学，在大三在发表了四大顶刊的论文，ICPR。然后之后保研PKU。

可能也是那时我发现，我如果考上了应该可以追平一点与他们的差距吧。

  • 展望

但是失败了又能怎样，我确实不是一个一蹶不振的人。

那么爬起来，至少离自己的梦想再近一点好不好？

2019 实习争取去一家有关人工智能的公司

2019 7月份再次开始复习考研，如果有工作的话，就闲暇时间利用好。

2019考研成功

2020-2022 读研两年 这时我应该25岁

2022-2025 一定要进好公司，三年我的目标就是赚钱。

2025-2027 如果我真的达到了以上的目标，我想用我自己存起来的钱创业。利用我自己学习的知识创业。

以上是大计划，小计划我会详细写。
每次一写这种文章会发现，自己还是太弱了。所以快点成长起来吧，成为一个优秀的人吧。请你一定要实现自己的梦想。

我的人生是为了自己自己活得呀。
